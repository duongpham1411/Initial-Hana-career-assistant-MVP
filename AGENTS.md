# JobBuddy Voice Agent - Coding Instructions

You are building a PC-first MVP for a voice-based AI career assistant.

Product name:
JobBuddy Voice Agent

Main goal:
Build a web app that helps job seekers research companies, analyze job descriptions, compare job fit, and receive short spoken responses from a female AI assistant.

Target user:
A student or fresher looking for Data Analyst / Business Analyst internships.

Tech stack:
- Frontend: React + Vite
- Backend: FastAPI
- Database: SQLite
- Styling: simple modern CSS, no heavy UI framework at first
- AI integration: use environment variables; do not hardcode API keys
- Voice: browser microphone for input, backend endpoints prepared for speech-to-text and text-to-speech

MVP screens:
1. Voice Chat
2. Company Research
3. JD Analyzer
4. Job Tracker
5. User Profile

Coding rules:
- Keep code readable and beginner-friendly.
- Use clear folder structure.
- Add comments only where useful.
- Create mock services first if API keys are not available.
- Never expose secrets in frontend.
- Add README with setup and run instructions.
- After every major task, summarize changed files.
